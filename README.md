**A fast and lightweight disk space analyzer for Windows.**

c² flux is a refactored continuation of [WTF – Where's the Filespace](https://github.com/UncleRiot/WTF), now developed as a separate project and repository.

## Highlights

- Complete UI overhaul
- Fast drive and folder scanning
- Global file and folder search
- Scan history and comparison
- Storage usage history
- Multiple analysis views
- CSV export
- Windows Explorer integration
- Light and dark themes
- German and English interface

## Requirements

- Windows 7 or newer
- Windows x64
- .NET 8 Desktop Runtime

> NTFS MFT scanning requires administrator privileges.
